## Task 1: Szkielet FS + i-węzły + katalogi (create/mkdir/lookup/readdir) + commit + persystencja
**Files:** Create `src/v2/gh2_fs.h`, `src/v2/gh2_fs.c`, `tests/test_v2fs.c`; Modify `gh2_btree.{c,h}`.
- [ ] `gh2_btree_iterate_range(dev, root, min_key, max_key, cb, ctx)` — in-order od ≥min do ≤max.
- [ ] `gh2_fs.h`: struct gh2_fs, gh2_inode, gh2_dirent, stałe typów (GH2_FT_*, GH2_INODE_ITEM=1,
  GH2_DIR_ITEM=2, GH2_EXTENT_DATA=3, GH2_SYMLINK_DATA=4), deklaracje API. `next_ino` w superbloku
  (dodaj pole / użyj reserved[0]).
- [ ] `gh2_fs.c`: kodowanie/dekodowanie inode item; FNV-1a 64-bit nazwy; dir_item pack/unpack
  (lista wpisów, kolizje hash); `gh2_fs_format` (gh2_format + root ino 1 DIR nlink2 + commit);
  `gh2_fs_mount` (gh2_mount + mark-sweep build mapy z fs_root + wczytaj next_ino) / `gh2_fs_unmount`;
  `gh2_fs_commit` (fsync danych → sb.root_tree=fs_root + gh2_commit_super → txn_alloc_commit);
  `gh2_path_resolve`; `gh2_fs_getattr`; `gh2_fs_create`/`gh2_fs_mkdir` (alokuj ino, INODE_ITEM,
  DIR_ITEM w rodzicu, nlink rodzica; -EEXIST); `gh2_fs_readdir` (range-scan DIR_ITEM + .,..).
  Każda mutacja: txn alloc vtable → gh2_btree_insert/delete → nowy fs_root; przy błędzie
  txn_alloc_abort + porzuć nowy root.
- [ ] Test (test_v2fs.c, +stdlib.h): format/mount/root; create/mkdir/lookup/getattr; zagnieżdżone;
  -EEXIST/-ENOENT/-ENOTDIR; readdir z wieloma wpisami (kolizje hash); **persystencja**: operacje
  → commit → unmount → remount → stan + mapa spójne; wyciek bloków = 0.
- [ ] `make build/test_v2fs && ./build/test_v2fs` 0 failed; ASan; regresja `make test` 0 failed. Commit.

- [ ] `gh2_fs_unlink` (nlink--, usuń INODE+SYMLINK_DATA przy 0, -EISDIR); `gh2_fs_rmdir`
